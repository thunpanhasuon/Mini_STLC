# CS67lib

**MINI_STLlibc** is a C library inspired by me and designed for students like me at RUPP to **practice memory management and low-level programming concepts**. It provides hands-on tools for working with dynamic arrays, strings, and linked lists, helping me to understant memory management.I'm also plan to add more feature like important dsa in computer science, and also add a human wirtten test becuase test and build is being generated by AI :(
